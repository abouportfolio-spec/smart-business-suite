# 🚀 Smart Business Suite

Une solution de gestion commerciale robuste et modulaire conçue avec **Django 5** et **Tailwind CSS**, déployée sur **PythonAnywhere**.

## 🏗️ Architecture du Projet
Ce projet suit une architecture découplée pour garantir une maintenance facile et une scalabilité optimale :
* **Core/** : Configuration centrale et sécurité.
* **Business/** : Module métier (Produits, Ventes, Dashboard).
* **Theme/** : Intégration Tailwind CSS pour une UI moderne.

## 🛠️ Stack Technique
* **Backend** : Python 3.12 / Django
* **Frontend** : Tailwind CSS
* **Hébergement** : Déploiement optimisé sur **PythonAnywhere**.

- ✅ Dashboard analytique des ventes.
- ✅ Gestion d'inventaire complète.
- ✅ Interface d'administration sécurisée.
